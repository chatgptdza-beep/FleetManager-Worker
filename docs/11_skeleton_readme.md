# Skeleton Readme

تم توسيع الـ skeleton ليشمل ربطًا أوضح بين:
- VPS nodes
- managed accounts
- stage timeline
- active alerts
- Desktop dashboard
- backend API

- قائمة nodes
- قائمة accounts مفلترة حسب الـ node المحددة
- panel تفاصيل الحساب والمرحلة المتأثرة بالتنبيه
